# Telegram Bot - Многофункциональный бот для управления чатами

## Обзор

Многофункциональный Telegram бот с 150+ командами, системой модерации, играми, внутренней валютой (звёзды и виртуны) и "Троллинг консолью" (премиум за 200⭐).

## Владелец

@n777snickers777 - полный доступ ко всем функциям бота

## Технологии

- **Framework**: Mastra (TypeScript)
- **База данных**: PostgreSQL
- **Обработка событий**: Inngest
- **Триггер**: Telegram Webhook

## Структура проекта

```
src/
├── mastra/
│   ├── agents/
│   │   └── telegramBotAgent.ts    # Агент бота
│   ├── tools/
│   │   ├── botCommands.ts         # 150+ команд бота
│   │   └── database.ts            # Функции работы с БД
│   ├── workflows/
│   │   └── telegramBotWorkflow.ts # Workflow обработки сообщений
│   ├── storage/
│   │   └── index.ts               # PostgreSQL storage
│   └── index.ts                   # Главный файл Mastra
├── triggers/
│   └── telegramTriggers.ts        # Обработчик Telegram webhook
```

## Таблицы базы данных

- `bot_users` - Пользователи (звёзды, репутация, уровень, био, префикс, виртуны, XP)
- `bot_chats` - Чаты (настройки, приветствия, правила, лимиты медиа)
- `global_users` - Глобальные пользователи (владельцы, премиум, виртуны)
- `warnings` - Предупреждения
- `temp_restrictions` - Временные баны/муты
- `blacklist_words` - Чёрный список слов
- `subscriptions` - Подписки (Троллинг консоль - 200⭐)
- `shop_prefixes` - Магазин префиксов (15 штук)
- `user_prefixes` - Купленные префиксы
- `message_stats` - Статистика сообщений
- `star_transactions` - Транзакции звёзд
- `reputation_history` - История репутации
- `reports` - Жалобы
- `payments` - Платежи

## Основные системы

### Экономика (9 команд)
- `/daily` - ежедневная награда 50-100 ⭐
- `/weekly` - еженедельная награда 300-500 ⭐
- `/pay @юзер [сумма]` - отправить звёзды
- `/transfer @юзер [сумма]` - трансфер звёзд
- `/top_rich` - топ богачей чата
- `/virtas` - показать виртуны
- `/buyvirtas [10]` - покупка 10k виртов за 10 ⭐

### Игры (6 команд)
- `/roll` или `/dice` - кубик 🎲
- `/casino [ставка]` - казино
- `/slots [ставка]` - слоты 🎰
- `/fish` - рыбалка 🎣
- `/duel @юзер` - дуэль ⚔️
- `/coin` - монета 🪙

### Брачная система (3 команды)
- `/marry @юзер` - предложение брака 💍
- `/accept_marry` - принять предложение 💕
- `/divorce` - развод 😢

### Троллинг консоль (Премиум 200⭐)
- `/smeshnoy_text` - смешные фразы на 6 часов
- `/kloun` - статус клоуна на 1 час
- `/unmuteall` - размут везде
- `/transform` или `/превратить` - трансформация в 7 образов
- `/invisibility` - невидимость

### RP-команды (36 текстовых команд без слеша)

**Боевые:** ударить, убить, выстрелить, зарезать, отравить, взорвать, сжечь, задушить, толкнуть, пнуть, связать, арестовать, обезглавить, расстрелять

**Позитивные:** обнять, целовать, погладить, улыбнуться, подмигнуть, пожать, утешить, похвалить, танец, комплимент, ужин, цветы, серенада

**Эмоции:** смеяться, плакать, вздохнуть, нахмуриться, удивиться, испугаться, разозлиться, восхититься, усмехнуться

**Физические:** бежать, спрятаться, замереть, присесть, лечь, встать, прыгнуть, нырнуть, кивнуть

**Магия:** заморозить, поджечь, ослепить, молния, проклятие, снять, исцелить, воскресить

### Команда дня
- `кто сегодня @юзер [текст]` - предсказание с рандомными ответами 🎰

### Модерация (25+ команд)
/ban, /softban, /tempban, /unban, /mute, /tempmute, /unmute, /ro, /unro
/warn, /unwarn, /warns, /resetwarns, /warnlimit
/kick, /kickme, /restrict, /unrestrict
/antispam, /flood, /blacklist, /whitelist, /caps, /links, /badwords
/media_limit, /promote, /demote, /clean, /pin, /unpin

### Информация и статистика
/profile, /balance, /id, /info, /stats, /top_rich, /chat_info, /users, /admins, /mods

### Команды владельца
- `/addcoins @юзер` - пополнить баланс до 9,999,999⭐
- `/givepremium @юзер [месяцы]` - выдать Троллинг консоль
- `/givestars @юзер [количество]` - выдать звёзды

## Монетизация

- **Внутренняя валюта**: Звёзды ⭐ и Виртуны (10k за 10⭐)
- **Ежедневный бонус**: 50-100 ⭐ (/daily)
- **Еженедельный бонус**: 300-500 ⭐ (/weekly)
- **Магазин префиксов**: 15 префиксов (100-5000 ⭐)
- **Троллинг консоль (Премиум)**: 200⭐/месяц

## Настройка Webhook

После публикации на Render установите webhook:
```
https://api.telegram.org/bot<TOKEN>/setWebhook?url=https://<YOUR_DOMAIN>/webhooks/telegram/action
```

## Примечания

- Текстовые RP-команды работают без слеша (например: "ударить @юзер")
- Для работы требуется упоминание пользователя (@username)
- Все команды с премиум требуют Троллинг консоль (200⭐/месяц)
- Кулдауны (КД) для смешного текста и клоуна - 6 часов
- Бот может работать на Render без проблем
